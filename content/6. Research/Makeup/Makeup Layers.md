# **PIEL (complexion)** 
___
<img
  loading= "lazy" 
  src="../media-content/Apuntes/media-moda/Makeup-Layers/Face Sculpting 3.jpg" 
  alt="Base"

• **Base / foundation** → unificación del tono
• **Skin finish** → acabado visual (mate, dewy, glass skin, soft matte)
• **Contouring** → esculpir volúmenes con sombra  
• **Highlighting** → puntos de luz estructurales  
• **Blush placement** → colocación del rubor (alto, bajo, centrado, “sunburn”)  

# **OJOS (eye makeup)**
___
<img
  loading= "lazy" 
  src="../media-content/Apuntes/media-moda/Makeup-Layers/Eye Zones.avif" 
  alt="Eyes"
  class="entry-image"

• **Eyeliner** → línea que define el ojo (clásico, cat eye, graphic liner, tightlining, Waterline color)  
• **Eyeshadow look** → combinación de sombras (cut crease, smoky, halo eye, gradient, definido/difuso, Topper shimmer, Glitter glue)  
• **Lash styling** → máscara + pestañas postizas (natural, doll, fox eye)  
• **Inner corner highlight** → punto de luz en el lagrimal  
• **Lower lash definition** → trabajo en la línea inferior (difuminado o marcado)
• Lower lash highlight.

# **3D Materials:**  
___
- Color base → color map (albedo)
- Brillo/gloss → roughness map
- Volumen visual (contour/highlight) → pintura de luces/sombras en textura
- Glitter →  specular map + normal suave
- Eyeliner y símbolos → máscaras de opacidad o decals
