# **Instalación Servicio FTP**

- Para instalar y configurar un servidor FTP con WS es necesario tener instalado previamente el servidor IIS.

- Para instalar el servidor FTP tenemos que instalar el rol de servidor que hace referencia al servidor FTP.

Vamos a Inicio → Administrador del servidor y pulsamos sobre Agregar roles y características

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image4.png"
style="width:4.34237in;height:2.46327in" />

Instalación basada en características o en roles y Siguiente

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image3.png"
style="width:5.90556in;height:4.17708in" />

Seleccionamos el servidor y Siguiente

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image6.png"
style="width:5.90556in;height:4.18819in" />

Desplegamos Servidor Web (IIS), seleccionamos Servidor FTP y Siguiente:

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image1.png"
style="width:5.90556in;height:4.18542in" />

Dejamos las características por defecto y Siguiente

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image2.png"
style="width:5.90556in;height:4.18056in" />

Confirmamos las selecciones de instalación pulsamos Instalar.

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image7.png"
style="width:5.90556in;height:4.18819in" />

Esperamos a que finalice la instalación y cerramos

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image5.png"
style="width:5.90556in;height:4.19514in" />

Confirmamos que se ha instalado correctamente desde el panel del Administrador

del Servidor:

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media FTP WIND/image8.png"
style="width:5.37575in;height:2.66704in" />
