# **PROTOCOLOS:**
___
>[!Info] **Protocolos**
>- **DHCP: (Dynamic host Configuration Protocol)** Protocolo de configuración dinámica de parámetros de red
>
>- **DNS: (Domain Name Service)** Protocolo de resolución de nombres de dominio
>- **FTP: (File Transfer Protocol)** Protocolo de transferencia de archivos
>- **UDP (User** **Datagram****~~Datagrama~~** **Protocol)**
>- **TCP: (Transmission Control Protocol)**
>- **IP (Internet Protocol)**
>- **HTTP: (HyperText Transfer Protocol)**
>- **IMAP: (Internet Message Access Protocol)**
>- **POP: (Post Office Protocol)**
>- **SMTP (Simple Mail Transfer Protocol)**
>- **SSL (Secure Sockets Layer)**
>- **SSH (Secure Shell)**

# SERVICIOS
___
>[!Info] **Servicio de configuración dinámica (DHCP)**
>- **Los servidores DHCP utiliza los Protocolos TCP/UDP (67 y 68)**
>- **Servicio de resolución de nombres (DNS)**
>- **Los servidores DNS utilizan TCP y UDP, en el puerto 53**

>[!Info] **Servidor de transferencia de ficheros**
>
>- **Protocolo FTP (20 y 21)**
>- **Protocolo FTPS (21 y 4460-4500)**
>- Protocolo TFTP (69)

>[!Info] **Servicios WEB**
>
>- **Protocolo HTTP (80)**
>- **Protocolo HTTPS (443)**

>[!Info] **Servicio de Acceso remoto**
>
>- Protocolo TELNET
>- **Protocolo SSH (22)**
>- Protocolo RDP (3389)
>- Protocolo RFB

>[!Info] **Servicio correo electrónico**
>
>- **Protocolo POP3 (110)**
>- Protocolo POP3s (995)
>- **Protocolo IMAP (143)**
>- Protocolo IMAPS (993)
>- **Protocolo SMTP (587)**
>- Protocolo SMTPS (465)
