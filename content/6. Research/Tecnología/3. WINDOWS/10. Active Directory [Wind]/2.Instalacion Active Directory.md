# Instalación **Active Directory**
___
Vamos a Inicio → Administrador del servidor y pulsamos sobre Agregar roles y características
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image1.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Instalación basada en características o en roles y Siguiente
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image11.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Seleccionamos el servidor y Siguiente
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image6.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Seleccionaremos el rol Servicios de Dominio de Active Directory:
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image19.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image3.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Dejamos las características por defecto y siguiente
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image12.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Leemos la descripción y observaciones del servicio y siguiente
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image16.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Confirmamos las selecciones de instalación pulsamos Instalar.
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image18.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Esperamos a que finalice la instalación y cerramos
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image15.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Ejecutamos el asistente, y las primeras pantallas nos ofrecen información útil para la creación del dominio
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image2.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Seleccionamos Agregar un nuevo bosque. E introducimos el nombre del dominio raíz que queremos crear
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image4.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

En el apartado “Opciones del controlador de dominio” debemos especificar el nivel de funcionamiento del bosque y del dominio (Sistema operativo), las capacidades del sistema de nombres de domino, en este caso Resolución de nombres de dominio y por ultimo establecer una contraseña para el administrador de Active Directory (Nivel funcionamiento, debemos indicar a partir de que versión funcionara)
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image7.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

En este apartado crea una delegación de servicio, le damos a siguiente y active Directory la creara solo. (Delegación DNS, es una configuración DNS)
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image14.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

Al empezar la configuración creamos un dominio de control para active Directory (Prueba.com)

En este apartado debemos indicar nuestro dominio, que en este caso es “Prueba”
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image8.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

En el apartado “Rutas de Acceso” debemos indicar donde va a almacenar active Directory sus bases de datos, por lo general dejaremos las por defecto
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image20.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image5.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

En el apartado “Revisar Opciones” nos muestra un resumen de la configuración al promover el controlador de dominio, nombre del nuevo dominio, nombre NetBIOS, nivel funcional del bosque y dominio, opciones adicionales en nuestro caso Servidor DNS, las comprobamos y siguiente.
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image13.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

En el “apartado de requisitos previos” nos hará un resumen de las comprobaciones de requisitos, ya sean configuraciones que hacen falta para la instalación o se han configurado mal, solucionamos en caso de ser necesario e instalamos, por lo general es posible que aunque se haya configurado bien haya algunas alertas, en un principio no hay problema, instalamos.
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image9.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

En caso de que la instalación se haya completado correctamente será necesario reiniciar la máquina para terminar la instalación.
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image10.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;"/>

Para volver a iniciar sesión debemos introducir la contraseña establecida durante la promoción a controlador de dominio.
<img src="/media-content/Apuntes/media-informatica/Media Windows/Media Active Directory WIND/image17.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />
