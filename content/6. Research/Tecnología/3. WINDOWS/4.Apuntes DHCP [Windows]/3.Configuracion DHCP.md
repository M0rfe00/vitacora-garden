# **Configuración Servicio DHCP**

**Antes de configurar el servicio debemos asignar un IP al servidor, para ello vamos al apartado: Servidor local -\> Ethernet -\> Ethernet -\> Protocolo IPv4**

**Y una vez dentro marcamos la casilla “Utilizar la siguiente dirección IP” para establecer una dirección IP estática al servidor e introducimos la IP que le queremos asignar con su respectiva mascara de red, enrutador y DNS en caso de ser necesario y pulsamos en validar al salir y aceptamos.**

**Es posible que salte un diagnóstico para detectar posibles problemas tras la configuración, cerramos y continuamos.**

**Una vez instalado procedemos a su configuración desde el apartado herramientas -\> DHCP**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image5.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Una vez dentro de la configuración DHCP accedemos al servidor que queremos configurar y accedemos al apartado IPv4**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image6.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Dentro de este panel hacemos clic derecho sobre IPv4 y accedemos a Nuevo ámbito y accederemos al asistente para la configuración**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image7.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**El primer apartado no pedirá que introduzcamos un nombre identificativo para la configuración**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image8.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado de Intervalo de direcciones IP direcciones, debemos indicar el intervalo de direcciones IP de nuestra red que queremos que asigne nuestro servidor a los clientes que la soliciten en este caso desde el primer host hasta el último, pero podría ser inferior**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image9.png" style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado exclusiones podemos indicar en caso de ser necesario las direcciones IP o intervalo que no queremos que asigne el servicio DHCP ya sea porque están reservadas a un cliente o no queremos que se usen por cualquier motivo, en este caso introducimos la IP
192.158.20.129**

**Ya que esta asignada al servidor y la hemos introducido en el intervalo de direcciones, al hacer esta exclusión hacemos que el servidor no asigne esa dirección IP para que no haya conflictos con el servidor**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image10.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado concesiones debemos introducir el tiempo el tiempo durante el que estarán asociadas las direcciones IPs a los clientes**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image11.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**El siguiente apartado nos indica que el ámbito que hemos configurado invalida las opciones predeterminadas del servidor modificándolas por las configuradas y para que se aplique hay que activar el ámbito, por lo que seleccionamos la configuración “Configurar estas opciones ahora”:**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image12.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado Enrutador o puerta de enlace debemos introducir la dirección IP del equipo de dará salida a nuestra red a otras, en este caso no vamos a salir a otras redes por lo que dejamos este campo en blanco y continuamos**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image13.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado nombre de dominio y Servidores DNS debemos introducir la IP del servidor que proporcionara la resolución de nombre de dominio a nuestra red.**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image14.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado Servidores WINS es igual al DNS y único de Windows, al haber rellenado la información DNS no introducimos nada aquí, (No lo usamos)**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image15.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**En el apartado Activar Ámbito no dice que los clientes solo podrán obtener concesiones si el ámbito esta activado y pregunta si queremos activarlo, marcamos activar ahora y finalizamos.**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image16.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Para crear una reserva nos dirigimos al ámbito anteriormente creado y accedemos a Reservas y hacemos clic derecho y seleccionamos la opción reserva nueva**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image17.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Esto nos abrirá la configuración para crear reservas, en la cual debemos de introducir la dirección IP que queremos asignar al cliente y su dirección MAC sin espacios ni guiones y un nombre para identificar la reserva**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image18.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Para comprobar que la reserva se ha aplicado correctamente refrescamos la conexión desde el apartado dispositivos -\> Red -\> desconectando y conectando el adaptador de red de la máquina virtual**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image19.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Para conocer la dirección MAC de nuestro cliente:**

**Lubuntu:**

**Para ver la dirección MAC en un sistema operativo Lubuntu vamos a las opciones de red en la parte inferior derecha de la barra de tareas y accedemos a la información de la conexión**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image20.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Esto nos abrirá un panel de información de las conexiones, debemos fijarnos en la dirección hardware, esta es la dirección MAC que debemos introducir en la reserva**

<img src="/media-content/Apuntes/media-informatica/Media Windows/Media DHCP WIND/image21.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

equipo entre ellas la dirección MAC**
