# **Posibles errores y soluciones** 
___
### **EL cliente y el servidor no tienen conexión:**
___
- El firewall esta activado
- El adaptador de la maquina esta en NAT
- El cliente está en manual/automática
- El direccionamiento está mal hecho (Mascara, Rango, puerta enlace…)



### **El navegador o explorador de archivos no accede a al sitio web(http) o sitio (FTP)**
___
- El DNS no está bien configurado
- El sitio web este detenido
- El documento predeterminado HTML esta abajo
- El host/sitio está mal asociado al fichero/web
