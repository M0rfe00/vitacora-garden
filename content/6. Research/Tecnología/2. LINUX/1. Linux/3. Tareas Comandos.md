1)  **Muestro únicamente los alumnos del archivo cuyo nombre empiece por “M”**
- Grep “^M” datos.txt
> <img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image4.png"
> style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

1)  **Muestro únicamente los alumnos del archivo cuya ciudad sea Sevilla o Badajoz**
- grep -E “Sevilla\|Badajoz” dato.txt
**-E** nos permite introducir varios patrones de búsqueda

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image8.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

3)  **Muestro los alumnos ordenados de mayor a menor según su edad:**
- tail -n +2 datos.txt \| sort -t”;” -k4.4nr datos.txt

**tail -n +2:** muestra todas las líneas a partir de la segunda ignorando la primera

**–t”,”:** indica que el delimitador entre los campos es “;”

**-k4,4nr:** indica que el campo en el que nos vamos a fijar es el 4º y ordenaremos forma numérica y con r de forma descendente

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image5.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

4)  **Muestro los alumnos ordenados de menor a mayor según su edad, sin la cabecera volcando el resultado en un nuevo archivo llamado ordenados.txt:**

- Sort -t”;” -k4.4nr datos.txt \> ordenados.txt

En este caso al no poner la variable **r** en **n**, ordenara de menor a mayor por defecto

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image12.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

5)  Muestro únicamente las localidades de cada alumno de forma ordenada

- Awk -F”;” “(print \$2)” datos.txt

**-F”;”:** especificamos cual es el delimitador entre los campos

**Print \$2:** especificamos que muestre el segundo campo

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image10.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

6)  **Calcúlo el número de alumnos que están en SMR2:**

> **-c:** cuenta el número de líneas que coinciden con el patrón de búsqueda
<img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image7.png" style="margin: 0 auto; padding: 0; width: 100%; height: auto; />

7)  **Cuenta el número de líneas del fichero datos.txt**
- wc -l datos.txt
**wc** (Word count)
**-l:** lineas

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media Linux Ubuntu/image3.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

8)  muestro el contenido numerando las líneas

8)  **Muestro únicamente los alumnos cuya edad este entre 20 y 29 años**
- awk -F';' '\$4 \>= 20 && \$4 \<= 29' datos.txt
**-F”;”:** especifica el delimitador
**\$4 \>= 20 && \$4 \<= 29:** filtra las líneas en las cuales el cuarto campo (edad) es mayor o igual a 20 y menor o igual a 29
