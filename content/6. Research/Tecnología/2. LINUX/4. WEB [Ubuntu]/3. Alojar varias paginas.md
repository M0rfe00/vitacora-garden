# **Alojar mas una pagina web con diferentes dominios**
___
**En el mismo archivo que creamos las zonas directas deberíamos añadir otra zona directa para el siguiente dominio, el cual apuntara a la misma zona indirecta**

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image1.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Al nuevo dominio que hemos creado, deberemos crearle un archivo “db.” Igual que al anterior archivo, con sus hosts y todo lo necesario**

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image2.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Dentro del mismo archivo de la zona inversa, añadiremos otro NS y PTR para el otro dominio igual que para el anterior**

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image3.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Evidentemente deberemos tener dos paginas web, para ello lo recomendable es crear una carpeta para cada index**
<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image4.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Después en la carpeta sites-avialable deberemos crear un archivo .conf para cada página web Importante: es obligatorio añadir una línea ServerName “dominio” para especificar a que dominio está vinculado esa página web y recomendable comentar la línea “ServerAdmin”**

**Ejemplo:**
<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image5.png" style="margin: 0 auto; padding: 0; width: 100%; height: auto;"/>

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image6.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Por último, creamos los enlaces simbólicos para cada uno de los archivos anteriormente creados**
<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image7.png"
style="margin: 0 auto; padding: 0; width: 100%; height: auto;" />

**Reiniciamos los servicios y comprobamos desde el navegador de un cliente que al introducir cada dominio nos resuelve con la página web correspondiente**

<img src="/media-content/Apuntes/media-informatica/Media Linux/Media WEB Ubuntu/image8.png"
