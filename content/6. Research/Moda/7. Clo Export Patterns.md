**Como preparar los patrones de Clo3D para imprimir**

# First Method
___
## Calibrar impresora
**Antes de importar las piezas del patron hay que calibrar la impresora, el tamaño del artboard en illustrator ha de ser el real, para ello hay que:**

1) Crear un rectángulo en Clo de 260x180mm y distribuir líneas internas para que cada segmento mida 20mm

/>

/>


2) Exportar como PDF, crear un artboard de 210mm x 297mm (A4)  en Illustrator e importar el artboard

<img
  loading= "lazy" 
  src="../media-content/Apuntes/media-moda/Prepare-Patterns/Pasted image 20260113204342.png"
  alt="Estructuras"

3) Exportar el proyecto como artboard

<img
  loading= "lazy"

  alt="Estructuras"
  style="margin: 0 auto; padding: 0; width: 100%; height: auto;"
/>

4) Imprimir el artboard y medir exactamente la dimensión total

<img
  loading= "lazy" 
  src="../media-content/Apuntes/media-moda/Prepare-Patterns/Pasted image 20260113204524.png"
  alt="Estructuras"
  style="margin: 0 auto; padding: 0; width: 100%; height: auto;"
/>


5) Dividir 26 entre el valor resultante, Ejemplo: 26 / 23,6 = 1,1017. Este valor hay que utilizarlo para crear un nuevo artboart calibrado en Illustrator, para ello hay que dividir 210 entre el valor anterior.

  alt="Estructuras"

/>


Al volver a imprimir tras haber calibrado la impresora el resultado deberia ser 26cm exactos

## Artboard
