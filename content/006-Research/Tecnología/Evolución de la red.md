# EVOLUCIÓN DE LA RED

  

### **WEB 1 (1991-2004)**

- Para estar en internet hay que tener una pagina WEB y para a internet es necesario un ordenador (los cuales eran fijos y escaseaban), Paginas web estáticas, solo lectura, sin identificación, publicidad generalizada, Usuarios = Consumidores

  
  

### **WEB 2 (2004-actualidad)**

- el internet va contigo a todas partes con la llegada de los teléfonos móviles,
- Internet centralizado en gigantes tecnológicos, las empresas recopilan datos de los usuarios, contenido centralizado, venta de datos a anunciantes, poca privacidad a cambio de contenido personalizado, redes sociales, móviles
- usuario = producto, conjunto de páginas y aplicaciones web

  
  

### **WEB 3**

- Red descentralizada, el usuario no es el producto, se accede a través de realidad virtuales, aumentadas, mixtas y asistentes virtuales, mundo en el que lo físico y digital se combinan, unificación de las plataformas web (métodos de pago, redes sociales, buscadores, chats, juegos, Netflix)

  
  

### **Internet de las cosas**

- Interconexión de todos los servicios web y los objetos comunes (coches, semáforos, lavadores, neveras, teléfonos, tv, calefacción, bombillas etc..)
