## 0) Búsqueda inspiración
- Buscar inspiración y referencias de modelos, prendas, desfiles, makeups, estilos, iluminaciones, fotografías, posters. 


## 1) DISEÑO PERSONAJE

>[!Info] DISEÑO PERSONAJE:
>- Añadir un personaje base de Genesis 9 y modificar los morphs
>- Añadir detalles de anatomía (poros, arrugas, venas, marcas de presión, uñas...)
>- Añadir pelo, pestañas y cejas.
>- Establecer pose y expresión.
>- Establecer materiales (Piel, Pelo, Ojos, Ropa...)
>- Añadir ropa (Zapatillas, ropa interior, accesorios...)

>[!Info] EXPORTAR PERSONAJE A SUBSTANCE PAINTER:
>- Exportar personaje en A Pose en "Base Resolution" (OBJ).
>- Exportar personaje posado en "Base Resolution" (OBJ).

>[!Question] LIMPIAR PERSONAJE EN BLENDER:
>- Importar el avatar en A Pose a Blender
>- Limpiar elementos innecesarios.
>- Eportar otro OBJ (Ej: Cabeza_Makeup.OBJ)

## 2) TEXTURIZADO MAKEUP/TATOO

>[!Abstract] DISEÑO TATOO PHOTOSHOP
>- Diseñar los tattoos en photoshop
>- Exportar en PNG o negro sobre blanco

>[!Abstract] MAKEUP CHART
>- Hacer moodboard con Base, Lips, Eyes, Shadow, Blush, Eyebrows, Hairstyle

> [!Success] TEXTURIZADO PERSONAJE EN SUBSTANCE PAINTER
> - Importar el OBJ del personaje en Substance.
>- Importar Diffuse y Normal.
>- Texturizar personaje (Makeup, Tattoos...). siguiendo las referencias y teniendo en cuenta materiales, técnicas, posibilidad de recreación, morfología facial, estilo...
>- Exportar texturas.

>[!Info] EXPORTAR PERSONAJE A BLENDER:
>- Guardar escena en una carpeta.
>- Exportar a Blender con Diffeomorphic guardando el DBZ en la misma carpeta.

## 3) BLENDER SETUP

>[!Question] LAYER PERSONAJE:
>- Crear una View Layer para el personaje.
>- Importar el personaje.
>- Optimizar separando por materiales y organizando en colecciones (Head, Body, Legs, Arms...).
>- Configurar texturas.

>[!Question] LAYER PELO:
>- Crear una View Layer para el pelo.
>- Organizar el pelo en una colección.
>- Configurar texturas.

>[!Question] LAYER ESCENARIO:
>- Crear una View Layer para el escenario.
>- Organizar los elementos del escenario en una colección.
>- Configurar texturas.

## 4) LIGHTNING & RENDER

>[!Question] ILUMINACIÓN Y CAMARA
>- Iluminar teniendo en cuenta la emoción que se quiere transmitir y el estilo.
>- Posicionar y configurar cámara según el plano.

>[!Question] CONFIGURAR RENDER
>- Configurar propiedades del render
>- Configurar propiedades del output

>[!Question] RENDERIZAR POR CAPAS
>- En el Compositor editor configurar las capas del render 
>- Renderizar Personaje
>- Renderizar Ropa
>- Renderizar pelo




## 5) POSTPROCESADO IMAGEN

>[!Abstract] EDITAR RENDER EN PHOTOSHOP
>- Ajustar luces
>- Ajustar colores
>- Aplicar efectos
>- Composición
>- ...
